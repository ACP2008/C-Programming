# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/ACP2008/C-Programming.git
```

### Open Folder

```bash
cd C-Programming
```

### Compile Program

Using GCC

```bash
gcc filename.c -o output
```

### Run

Windows

```bash
output.exe
```

Linux / macOS

```bash
./output
```
